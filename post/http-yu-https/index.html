HTTP与HTTPS

2021-04-07

HTTP

HTTPS

  1. 是什么
    HTTP+SSL(Security Sockets Layer)，通过SSL搭建隧道进行通信
  2. 为什么：HTTP存在安全性问题
    • 使用明文通信，容易被窃取
    • 无法确认通信方的身份，容易伪装
    • 报文可能会被修改
  3. 怎么办
    为了解决http的安全性问题，https通过以下三点来实现：
    • 加密（非对称加密与对称加密相结合）：首先使用非对称加密进行身份认证，认证成功后使用对称加密进行数据传输。
    • 认证（证书机制）：TSL证书核验
    • 完整性保护（报文摘要）：通过加密和认证机制进行完整性保护
